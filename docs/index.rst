Snippets is online reStructuredText and Sphinx editor with preview. Snippets are place to play and learn reStructuredText and Sphinx syntax. You write documentation in reStructuredText and Snippets render it with Sphinx.

This is the documentation of its frontend called *Snippets-web*. Backend counterpart is developed and documented separately as *Snippets-api* (https://github.com/documatt/snippets-api/).
